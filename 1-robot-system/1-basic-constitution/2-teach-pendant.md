# 1.1.2 티치 펜던트

티치 펜던트는 TP600과 TP630을 지원합니다. 본 조작 설명서에서는 _TP630_ 모델을 기준으로 사용 방법을 설명합니다.

TP630은 기존 Hi5a 제어기와 동일한 조작키 사용 환경을 제공합니다.
