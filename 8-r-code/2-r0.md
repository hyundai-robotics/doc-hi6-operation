# 8.2 R0 스텝 카운터 리셋

즐겨찾기 창에서 0을 입력한 후 \[확인\] 버튼을 터치하거나 &lt;enter&gt; 키를 누르십시오.

![](../_assets/image_271.png)

스텝 카운터를 초기화하여 STEP0으로 이동합니다. 또한 다음의 기능을 실행할 수 있습니다.

* 플레이백 실행 상태 클리어
* 종합 이상 신호, 램프 끄기
* 경보 신호 끄기
* WAIT 대기 상태 클리어
* 각종 응용 기능 상태 및 신호 클리어 등

{% hint style="info" %}
로봇 기동 중에는 R0 코드를 사용할 수 없습니다.
{% endhint %}
